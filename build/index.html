indrz Buildings API

This is our high-quality buildings API. You can use this API to request and remove different buildings.

List Buildings

Lists all buildings for a particular campus.

GET
/api/v1/{username}

Example request

$ curl https://www.indrz.com/api/v1/{username}

Example response

[
  {
    "owner": "{username}",
    "id": "{building_id}",
    "created": "{timestamp}",
    "modified": "{timestamp}"
  },
  {
    "owner": "{username}",
    "id": "{building_id}",
    "created": "{timestamp}",
    "modified": "{timestamp}"
  }
]

Create building

Creates a new, empty building.

Property Description
name (optional) the name of the building
description (optional) a description of the building
POST
/api/v1/{username}

Example request

curl -X POST https://www.indrz.com/api/v1/{username}

Example request body

{
  "name": "foo",
  "description": "bar"
}

Example response

{
  "owner": "{username}",
  "id": "{building_id}",
  "name": null,
  "description": null,
  "created": "{timestamp}",
  "modified": "{timestamp}"
}

Retrieve a building

Returns a single building.

Retrieve information about an existing building.

GET
/api/v1/{username}/{building_id}

Example request

curl https://www.indrz.com/api/v1/{username}/{building_id}

Example response

{
  "owner": "{username}",
  "id": "{building_id}",
  "created": "{timestamp}",
  "modified": "{timestamp}"
}

Update a building

Updates the properties of a particular building.

Property Description
name (optional) the name of the building
description (optional) a description of the building
PATCH
/api/v1/{username}/{building_id}

Example request

curl --request PATCH https://www.indrz.com/api/v1/{username}/{building_id} \
  -d @data.json

Example request body

{
  "name": "foo",
  "description": "bar"
}

Example response

{
  "owner": "{username}",
  "id": "{building_id}",
  "name": "foo",
  "description": "bar",
  "created": "{timestamp}",
  "modified": "{timestamp}"
}

Delete a building

Deletes a building, including all wibbles it contains.

DELETE
/api/v1/{username}/{building_id}

Example request

curl -X DELETE https://www.indrz.com/api/v1/{username}/{building_id}

Example response

HTTP 204

List buildings

List all the buildings in on a campus. The response body will be a buildingCollection.

GET
/api/v1/{username}/{building_id}/campus

Example request

curl https://www.indrz.com/api/v1/{username}/{building_id}/campus

Example response

{
  "type": "building",
  "campus": [
    {
      "id": "{campus_id}",
      "type": "building",
      "properties": {
        "prop0": "value0"
      }
    },
    {
      "id": "{campus_id}",
      "type": "building",
      "properties": {
        "prop0": "value0"
      }
    }
  ]
}

Insert or update a space

Inserts or updates a space in a building. If there's already a space with the given ID in the building, it will be replaced. If there isn't a space with that ID, a new space is created.

Property Description
id the id of an existing space in the building
PUT
/api/v1/{username}/{building_id}/campus/{space_id}

Example request

curl https://www.indrz.com/api/v1/{username}/{building_id}/campus/{space_id} \
  -X PUT \
  -d @file.geojson

Example request body

{
  "id": "{space_id}",
  "type": "building",
  "properties": {
    "prop0": "value0"
  }
}

Example response

{
  "id": "{space_id}",
  "type": "building",
  "properties": {
    "prop0": "value0"
  }
}

Retrieve a space

Retrieves a space in a building.

GET
/api/v1/{username}/{building_id}/campus/{space_id}

Example request

curl https://www.indrz.com/api/v1/{username}/{building_id}/campus/{space_id}

Example response

{
  "id": "{space_id}",
  "type": "building",
  "properties": {
    "prop0": "value0"
  }
}

Delete a space

Removes a space from a building.

DELETE
/api/v1/{username}/{building_id}/campus/{space_id}

Example request

curl -X DELETE https://www.indrz.com/api/v1/{username}/{building_id}/campus/{space_id}

Example response

HTTP 204

Geodata

Working with geodata such as points, linestrings and polygons is beyond the scope of the indrz project. If you are looking for help to work and convert your geodata data for indrz directions api please contact us at GOMOGI Communication is simple via email (sales@gomogi.com).

Georeferencing

All maps or plans must be georeferenced to work with indrz. This means you need to give real world coordinates to either your AutoCAD DWG files or images (png, jpeg, tiff, etc).

Open Source software tools

  1. Use [QuantumGIS] (http://www.qgis.org) and the Web map pluggin to view in google maps or satelite the building you want to georeference
  2. Once you have found the building click File > export image, this will export a georefence image from google maps
  3. Open this georeferenced image as a background to place your AutoCAD or other digital data
